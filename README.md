# Node.js CRUD API with MongoDB (MVC + EJS)

This project demonstrates how to build a CRUD app in Express.js using HTML templates, MongoDB, and an MVC code structure with basic error handling.

**What It Covers**
1. Set up HTML templates in Express.js (EJS views).
2. Read data from MongoDB.
3. Add, delete, update, and read data from an HTML form.
4. Structure code using the MVC pattern with error handling.

**Core Features**
1. Server-rendered HTML views with Express + EJS.
2. Full CRUD workflow connected to MongoDB.
3. Form-based create, update, delete, and read operations.
4. MVC-style separation of concerns.
5. Error handling for common request and database failures.

**Success Criteria**
1. EJS templates render properly in Express for all pages.
2. The app can read and display data from MongoDB.
3. Users can add, update, and delete records via HTML forms.
4. MVC structure is followed (controllers, models, views are separated).
5. Errors are handled gracefully without crashing the app.
